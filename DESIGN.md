# Design System: Secure Checkout

## Overview
The Secure Checkout design system is built for a premium, high-trust mobile payment experience. It focuses on clarity, security, and a vibrant, professional aesthetic.

## Typography
The project uses a modern, legible typeface designed for digital interfaces.

- **Primary Font Family**: Plus Jakarta Sans
- **Characteristics**: Geometric, sans-serif, high legibility for mobile devices.

## Color Palette
The color scheme is designed to feel energetic yet trustworthy, utilizing a warm accent color against a clean light background.

| Layer | Color | HEX | Description |
| :--- | :--- | :--- | :--- |
| **Primary** | Vibrant Orange | `#ec6d13` | Used for call-to-actions, primary buttons, and critical status highlights. |
| **Background** | Clean White | `#FFFFFF` | The base canvas for the application in Light Mode. |
| **Saturation** | Level 3 | N/A | Balanced saturation to maintain professional vibrance without being overwhelming. |

## Shape & Geometry
The interface utilizes soft, generous rounding to create a friendly and accessible user experience.

- **Corner Radius**: `12px` (ROUND_TWELVE)
- **Application**: Consistent across buttons, input fields, and containers to maintain a cohesive visual language.

## Design Principles
1. **Safety First**: Visual cues that reinforce security and trust.
2. **Clarity**: High contrast typography and clear primary actions.
3. **Modern Warmth**: Using `#ec6d13` provides a unique, energetic alternative to traditional "payment blue" while maintaining professionalism.
